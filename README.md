<h2 align="center">🚀 Featured Project</h2>

<div align="center">

  <h3>🛡️ StegoShield — Advanced Secure Steganography System</h3>

  <p style="max-width: 700px;">
    A web-based digital forensic platform for detecting and analyzing steganography
    and hidden malicious content in files. Built as my BS thesis project, it integrates
    multiple security and forensic tools into a single Flask-based dashboard.
  </p>

  <b>Tech stack:</b> Python · Flask · YARA · ClamAV · ExifTool · Binwalk · TensorFlow · SQLite<br><br>

  <b>Key features:</b><br>
  🔍 Automated steganography & hidden-data detection<br>
  🦠 Malware scanning via ClamAV and YARA rule matching<br>
  🧬 Metadata extraction and file analysis with ExifTool & Binwalk<br>
  🤖 ML-based anomaly detection using TensorFlow<br>
  🖥️ Dark, cybersecurity-themed dashboard UI with encryption/decryption workflows

</div>
